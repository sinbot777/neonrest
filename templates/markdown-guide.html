Markdown Guide

Neonrest supports simple Markdown formatting in posts.

Headings

# Heading 1
## Heading 2
### Heading 3

Bold and Italic

*italic* or _italic_
**bold** or __bold__
***bold italic***

Lists

- Item one
- Item two
  - Sub-item

1. First item
2. Second item

Links

[Link Text](https://example.com)

Images (coming soon)

![Alt text](image_url)

Code

Inline:

`inline code`

Block:

```
code block
multiple lines
```

Blockquotes

> This is a quote.
